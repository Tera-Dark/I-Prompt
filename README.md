## 🎯 项目简介

I-Prompt 是一款专业的AI绘画提示词助手，集成了**世界首个开源NovelAI Stealth PNG解析器**和**全新升级的智能提示词库3.0**，帮助艺术家和创作者快速生成高质量的英文提示词，并深度分析AI生成图像的元数据。通过集成硅基流动的DeepSeek-R1大模型、阿里云翻译API和先进的图像分析技术，提供从创作到分析的完整工作流。

### ✨ 核心特色

- 🤖 **AI智能生成** - 接入DeepSeek-R1大模型，提供专业级提示词生成
- 🔍 **专业图像分析** - 融合NovelAI Spell技术，支持所有主流AI工具
- 🎨 **智能提示词库3.0** - 全新界面设计，分层标签系统，阿里云翻译
- ✍️ **专业编辑器** - 双栏式编辑，实时翻译，智能补全
- 🌏 **中英文双语** - 完美支持中文输入，自动转换为英文标签
- ⚡ **降级保障** - 网络异常时自动使用本地增强算法
- 🎯 **精准分类** - 6大主分类，300+专业标签，20+子分类
- 📋 **一键操作** - 生成结果一键复制，快速分享

- **双栏布局**：英文和中文并排显示，方便对比

#### 🔧 技术特性

**支持的翻译引擎：**
- **MyMemory** (主推荐) - 支持330种语言，每日1000次免费调用
- **LibreTranslate** (开源) - 支持20种语言，开源免费
- **Google Web** (稳定) - 支持134种语言，非官方API
- **Bing Web** (实验性) - 支持128种语言，需要代理
- **Yandex Web** (实验性) - 支持102种语言
- **DeepL Web** (高质量) - 支持33种语言，质量最高

**智能特性：**
- 🧠 **智能引擎选择**：根据文本类型和可用性自动选择最佳引擎
- 🔄 **自动重试机制**：失败时自动切换引擎重试
- 📚 **专业词典降级**：在线翻译失败时使用内置AI绘画词典
